## Usage:

Run with no arguments to receive wisdom:

`java -jar target/wiseRat-1.0.jar`

Share the words of wisdom directly:

`java -jar target/wiseRat-1.0.jar "Wise words..."`

Pipe the wisdom from stdin:

`echo "Some other wise words..." | java -jar target/wiseRat-1.0.jar`
